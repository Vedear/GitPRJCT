## Скрипт обновления установленных приложений Windows, Linux и Mac

Задача: создать скрипт, который будет обновлять версии всех установленных в вашей ОС приложений.

- Для **Linux Ubuntu** команда обновления: `sudo apt list --upgradable -a && sudo apt update && sudo apt full-upgrade -y`;
- Для **Windows 10/11**: `winget upgrade --all`;
- Для **MacOS** (не проверено!): `sudo softwareupdate -ia`;
- Опционально можно сделать так, чтобы ваш скрипт выполнялся явно, в окне автоматически запустивнегося Терминала, для визуального контроля его выполнени;
- Протестировать скрипт. Если синхронизация прошла успешно, то можно поместить его в автозагрузку вашй ОС для удобства;
- Результат (код скрипта с описанием его работы) прислать в виде ссылки на вайл `README.md` с описанием проекта;
- Используйте **нейросети** для помощи с кодом.
